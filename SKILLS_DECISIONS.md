# Skills Decisions For Interview Online

Источник для анализа: [anthropics/skills](https://github.com/anthropics/skills/tree/main/skills).

## Применяем в проекте

1. `frontend-design`
- Нужен для улучшения визуальной целостности, композиции экранов и современных UI-паттернов.
- Применяем для личного кабинета, комнаты интервью и главной.

2. `theme-factory`
- Нужен для стабильной дизайн-темы (цветовые токены, контраст, согласованность компонентов).
- Применяем для единой темной темы Mantine и согласованных полей форм.

3. `webapp-testing`
- Нужен для быстрых e2e/smoke прогонов реальных пользовательских сценариев.
- Применяем для проверки комнаты, шагов интервью, модалки кандидата и кабинета.

4. `claude-api`
- Нужен как инфраструктурный скилл на случай интеграции внешних AI-фич.
- Для MVP не обязателен, но полезен как задел для генерации/оценки задач.

## Не применяем сейчас

1. `canvas-design`, `algorithmic-art`, `web-artifacts-builder`, `brand-guidelines`
- Не критично для MVP функционала собеседований.

2. `docx`, `pdf`, `pptx`, `xlsx`, `doc-coauthoring`, `internal-comms`, `slack-gif-creator`
- Не относятся напрямую к runtime-функционалу платформы.

3. `mcp-builder`
- Отложено до момента, когда потребуется отдельная MCP-интеграция.
